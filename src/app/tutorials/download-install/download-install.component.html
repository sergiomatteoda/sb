Introduction

In this tutorial you will learn to:
  • Download SendBlaster
  • Install SendBlaster

We will show how to download and install SendBlaster in Windows 10, however SendBlaster can be installed on any windows version.

Note: While SendBlaster is a windows based Bulk Emailer, it may still be installed on MacOs, following this article: Sendblaster on Mac? Here’s how it works

System requirements

  • Windows XP / Windows Vista / Windows 7 / Windows 8 / Windows 10
  • Microsoft Internet Explorer 8 or higher
  • 100 MB of available hard-disk space
  • 1 GB of RAM
  • Internet connection (broadband connection recommended)

Download SendBlaster

In order to Download SendBlaster, please navigate to the official download page, or use the bellow button.

Download for Free

Notice that SendBlaster is downloaded as a compressed zip file, it´s important that you extract the content of this file to a local folder.

In this example, the compressed zip file has been downloaded to "C:\SendBlaster" and the content (setup.exe) file has been extracted.

Install SendBlaster

After you have completed the previous step, you can see SendBlaster installer file setup.exe, double click on this file to start the install process.

Note: SendBlaster installer can be run under any regular Windows user account, however under rare Windows security configurations, it´s possible that the installer may require elevated priviledges.

If during installation you get any error that looks like something related to writting security, you may right click on setup.exe file and select the option to "run as administrator". If you still get issues on the installation, as usual, refere to Troubleshooting SendBlaster Installation.

Preparing to Install

After you executed the installer file setup.exe, the install wizard will be executed, it will prepare everything in your computer so SendBlaster can be installed. Wait for a few seconds...
