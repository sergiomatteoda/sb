Introduction

In this tutorial you will learn to:
  • Download SendBlaster
  • Install SendBlaster

We will show how to download and install SendBlaster in Windows 10, however SendBlaster can be installed on any windows version.

Note: While SendBlaster is a windows based Bulk Emailer, it may still be installed on MacOs, following this article: Sendblaster on Mac? Here’s how it works

System requirements

  • Windows XP / Windows Vista / Windows 7 / Windows 8 / Windows 10
  • Microsoft Internet Explorer 8 or higher
  • 100 MB of available hard-disk space
  • 1 GB of RAM
  • Internet connection (broadband connection recommended)

Download SendBlaster

In order to Download SendBlaster, please navigate to the official download page, or use the bellow button.

Download for Free

Notice that SendBlaster is downloaded as a compressed zip file, it´s important that you extract the content of this file to a local folder.

In this example, the compressed zip file has been downloaded to "C:\SendBlaster" and the content (setup.exe) file has been extracted.

Install SendBlaster

After you have completed the previous step, you can see SendBlaster installer file setup.exe, double click on this file to start the install process.

Note: SendBlaster installer can be run under any regular Windows user account, however under rare Windows security configurations, it´s possible that the installer may require elevated priviledges.

If during installation you get any error that looks like something related to writting security, you may right click on setup.exe file and select the option to "run as administrator". If you still get issues on the installation, as usual, refere to Troubleshooting SendBlaster Installation.

Preparing to Install

After you executed the installer file setup.exe, the install wizard will be executed, it will prepare everything in your computer so SendBlaster can be installed. Wait for a few seconds...

SendBlaster License Agreement

Before you can use SendBlaster, you´re required to understand and accept SendBlaster License Agreement.

In short, SendBlaster is owned by Delivery Tech Corp and protected by copyright laws and international copyright treaties, as well as other intellectual property laws and treaties.

We suggest you read the complete License Agreement, it´s not long.

After you´re done reading, please make sure you select the "I accept the terms in the license agreement" option, and hit "Next" button.

Note: You will allways be able to find the last updated version of SendBlaster License Agreement at: SendBlaster License Agreement.

Customer Information

This step will let you specify your "user name" and "organization", when available this details will be retrived from your windows installation. You may specify any details here, SendBlaster will not actually make use of this information.

You will also be required to specify wether SendBlaster will be available for "Anyone who uses this computer" or "Only for me". If there are several windows user accounts created in your computer, and you select the first option, then all windows users will be able to use SendBlaster, on the other hand, if you select the second option, only you will be able to use SendBlaster.

Note: If you have several windows user accounts and you want to provide access to SendBlaster to more than one windows account, but not "Anyone who uses this computer" select "Only for me" option, and repeat the entire installation process under each windows user account where you want to provide access to SendBlaster.

Note: While it´s possible to install SendBlaster under more than one windows user account, each installation will be treated as an independant installation, ie: if there´s a windows account named Joe, and another windows account namde Jane, then Joe and Jane will not share contact lists or any other SendBlaster data, each user will have a completly different set of user data.
